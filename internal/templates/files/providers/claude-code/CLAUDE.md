# CLAUDE.md

Use the Leaserage workflow. Prefer small safe changes, repository-local instructions, targeted verification, and closeout with summary, files changed, verification, and risks.

If RTK was initialized by Leaserage, its official hook rewrites supported shell commands automatically. If the hook is not active but `rtk` is installed, prefer `rtk <command>` for noisy checks and use `RTK_DISABLED=1` when raw output is required.
