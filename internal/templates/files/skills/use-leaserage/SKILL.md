# Use Leaserage

Use this as the bootstrap router for non-trivial work. Pick the smallest applicable workflow skill: brainstorming, debug, plan, implement, review, or test.

Repository-local instructions always win. Keep changes small, verify before completion, and report what changed, what was validated, and what remains risky.
